# Focus — Personal Productivity Dashboard

A simple productivity dashboard I built to practice vanilla JS. No frameworks, no build step — just HTML, CSS, and JS.

## What's in it

- **Tasks** — add tasks with priority and due dates, filter by status
- **Pomodoro timer** — 25/5 work-break cycles with browser notifications
- **Notes** — freeform scratchpad that auto-saves as you type
- **Daily quote** — pulled from an API, cached so it doesn't refetch every refresh
- **Stats** — tracks tasks done and focus sessions for the day
- **Dark mode** — respects your OS preference, toggleable

## How to run

Just open `index.html` in a browser. No install needed.

## Keyboard shortcuts

| Key | What it does |
|-----|-------------|
| `N` | Jump to task input |
| `P` | Start / pause the timer |

## Stack

Vanilla HTML · CSS custom properties · ES2020 JS · localStorage
